# AI-Powered Women Safety and Emergency Response System

## Overview
The AI-Powered Women Safety and Emergency Response System is a smart safety application designed to provide quick emergency assistance and improve personal security for women. The system helps users send emergency alerts, share live location details, and contact emergency services during unsafe situations.

This project focuses on real-time safety support and rapid emergency communication using modern web technologies.

---

## Features
- SOS Emergency Alert System
- Emergency Contact Notification
- Real-Time Location Sharing
- Quick Emergency Response Support
- User-Friendly Interface
- Secure Authentication System
- Instant Alert Trigger Mechanism

---

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- SQLite / Firebase

---

## Project Workflow
1. User logs into the system.
2. In emergency situations, the user can trigger the SOS alert.
3. The system sends emergency notifications to saved contacts.
4. Real-time location details are shared for quick assistance.
5. Emergency response support is activated instantly.
